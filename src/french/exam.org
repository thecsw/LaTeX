* Passé
** Passé Composé

*** Avoir

avoir + verbe(é) (manger-mangé, parler-parlé, finir-fini)

| j'    | ai    |
| tu    | as    |
| il    | a     |
| elle  | a     |
| nous  | avons |
| vous  | avez  |
| ils   | ont   |
| elles | ont   |

**** Exceptions

Avoir - eu

**** Other verbs

| ouvrir | ouvert |
| offrir | offert |

*** Être

| je    | suis   |
| tu    | es     |
| il    | est    |
| elle  | est    |
| nous  | sommes |
| vous  | etes   |
| ils   | sont   |
| elles | sont   |

**** Exceptions

| aller     | allé     |
| venir     | venu     |
| partir    | parti    |
| arriver   | arrivé   |
| monter    | monté    |
| entrer    | entré    |
| rester    | resté    |
| sortir    | sorti    |
| descendre | descendu |
| tomber    | tombé    |
| mourir    | mort     |
| naitre    | né       |
| passer    | passé    |
| tourner   | tourné   |

**** Warnings!

| partir | nous part *ons*   |
| finir  | nous fini *ssons* |
